# DailyProfilePictureTelegram
A telegram script that changes your profile picture depending on the day of the week. I did this mainly because i was bored and tought it would be funny to have 7 images that rotate daily.

## Use
You can setup in your pc with a service to run the program daily, but i sure dont wat to do that so lets setup a online vps to do this one time and then never have to bother with this again.
